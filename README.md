## Setup
This section shows the setup used for this project

### Ubuntu
1. Install Anaconda 3
2. Add to path
3. Install pandas-datareader module

### Necessary Python Modules
1. pandas (included in anaconda)
2. matplotlib (included in anaconda)
3. pandas-datareader
4. datetime
5. os
6. numpy
